🚀 快速开始

三步上手，立即体验智能测试用例生成

1️⃣

创建项目

在项目管理页面输入项目名称，选择目标测试框架（pytest/unittest/JUnit/TestNG）

2️⃣

配置场景

添加功能模块，配置输入参数（类型、范围、枚举），选择测试方法论

3️⃣

一键生成

点击生成按钮，自动产出测试用例，导出为 Excel / CSV / 测试代码

🏗️ 项目架构

清晰的模块化设计，易于扩展和维护

testcase_generator/
├── __init__.py          # 包声明
├── models.py            # 数据模型（Project / Scenario / TestCase / ParameterRule）
├── engine.py            # 测试方法论引擎（6大引擎 + 统一调度）
├── codegen.py           # 代码生成器（pytest / unittest / JUnit / TestNG）
├── exporter.py          # 多格式导出器（Excel / CSV / JSON）
├── theme.py             # 全局样式表（浅色/深色主题）
├── ui_sidebar.py        # 侧边栏导航组件
├── ui_dialogs.py        # 场景配置对话框 & 参数编辑器
├── ui_dashboard.py      # 工作台仪表盘
├── ui_project.py        # 项目管理页面
├── ui_scenario_manager.py # 场景管理页面
├── ui_case_manager.py   # 用例管理页面
├── ui_export.py         # 导出中心页面
├── ui_statistics.py     # 统计分析页面
└── ui_mainwindow.py     # 主窗口（集成所有页面）

main.py                  # 程序入口
build.bat                # 一键打包脚本
build.spec               # PyInstaller 配置
README.md                # 项目文档
